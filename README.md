## Problem Documentation Format

Each problem in this repository is documented in a structured and consistent format to ensure clarity and ease of understanding.

Each problem includes:

- **Problem Title**
- **HackerRank Problem Link**
- **Difficulty Level**
- **Problem Description**  
  A clear and concise explanation of the problem statement in simple terms.
- **Approach / Algorithm**  
  Step-by-step explanation of the logic used to solve the problem and the reasoning behind the chosen approach.
- **Code Explanation**  
  Detailed explanation of the Java implementation, including:
  - Key conditions and control flow  
  - Data structures used  
  - Important loops and logic decisions  
- **Java Solution Code**
- **Time Complexity**
- **Space Complexity**

This documentation approach supports both **conceptual understanding** and **interview-oriented revision**.
